AwoCoupon is an efficient and powerful coupon system for WooCommerce. It provides more features for discounting and includes automatic discounts.

== Description ==

AwoCoupon is a coupon sytem for WooCommerce with enhanced features.  It has an intuitive and simple to use interface and works well alongside WooCommerce coupons.

Major features of AwoCoupon

*   Setup automatic discounts 
*   Create shipping specific coupons
*   Create coupons with optional restrictions on product, category, customer, user group, country, state, payment method
*   Personalize coupon errors
*   Generate multiple coupons
*   Customers can view coupons assigned to them (or their user group) under my account in the front end

Compatibility

*   Tested on WooCommerce 3.0.0 and up
*   For multiingual sites, requires [Polylang](https://wordpress.org/plugins/polylang/) plugin
*   For user groups, requires [Groups](https://wordpress.org/plugins/groups/) plugin

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
